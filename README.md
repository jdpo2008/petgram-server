## Instalation

1. Clone this repo
```bash
git clone https://github.com/febryardiansyah/petgram-server.git
```

2. Install dependencies
```bash
npm install
```

3. Set up .env
Create file named `.env` in root directory and fill it with `.env.example`.

4. Run server
    - Production
    ```bash
    npm run start
    ```
    - Development
    ```bash
    npm run dev
    ```
